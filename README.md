## в—Ҳ PAYMENT SETUP (PAYSTACK)

**вҖ” Step 01 В· Create Account**
```
вҶ’ dashboard.paystack.com
вҶ’ Sign up with business email
вҶ’ Complete KYC (BVN + valid ID)
```

**вҖ” Step 02 В· Get Your Key**
```
Dashboard вҶ’ Settings вҶ’ API Keys & Webhooks
Copy:  pk_live_...   (live)
  or:  pk_test_...   (testing)
```

**вҖ” Step 03 В· Add It to Your Code**

```javascript
// In index.html or cart.html вҖ” find this line:
const PAYSTACK_KEY = 'pk_test_xxxxxxxxxxxxxxxxxxxxxxxx';

// Replace with your real key:
const PAYSTACK_KEY = 'pk_live_yourrealkeyhere';
```

**вҖ” Step 04 В· Link Your Bank**
```
Dashboard вҶ’ Settings вҶ’ Bank Account вҶ’ Add Account
Select your bank вҶ’ Enter account number
Paystack verifies вҶ’ payments auto-settle daily вң“
```

> рҹ”җ Rule #1 вҖ” **Never** expose your **Secret Key** in frontend code. Public Key only.
